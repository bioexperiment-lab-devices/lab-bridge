# lab-bridge documentation

This is the default landing page for your lab-bridge documentation portal.

Replace this file by uploading a new `index.md` through the admin panel
at [/admin/docs](/admin/docs), or by editing it directly on the VPS at
`site_data/docs/index.md`.

Drop additional `.md` files (and folders for sections) under `site_data/docs/`
to extend the portal. Each folder needs an `index.md` to act as the section
landing page. To add a Russian translation for any page, drop a sibling file
named `<name>.ru.md` next to it — an EN/RU toggle will appear automatically.
